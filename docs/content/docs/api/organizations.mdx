## Get Organization Sites

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/organizations/:organizationId/sites
```

Returns all sites within an organization along with session counts from the last 24 hours and subscription information. Requires membership in the organization.

### Path Parameters

<TypeTable
  type={{
    organizationId: {
      description: 'Organization ID',
      type: 'string',
      required: true,
    },
  }}
/>

### Response

<TypeTable
  type={{
    organization: {
      description: 'Organization details',
      type: 'object | null',
    },
    sites: {
      description: 'Array of sites in the organization',
      type: 'array',
    },
    subscription: {
      description: 'Subscription and usage information',
      type: 'object',
    },
  }}
/>

#### Site Object

Each site in the `sites` array contains:

<TypeTable
  type={{
    siteId: {
      description: 'Numeric site ID',
      type: 'number',
    },
    id: {
      description: 'Site UUID',
      type: 'string',
    },
    domain: {
      description: 'Site domain',
      type: 'string',
    },
    name: {
      description: 'Site name',
      type: 'string',
    },
    organizationId: {
      description: 'Organization ID',
      type: 'string',
    },
    createdBy: {
      description: 'User ID who created the site',
      type: 'string',
    },
    public: {
      description: 'Whether analytics are publicly accessible',
      type: 'boolean',
    },
    saltUserIds: {
      description: 'Whether user IDs are salted',
      type: 'boolean',
    },
    blockBots: {
      description: 'Whether bot traffic is blocked',
      type: 'boolean',
    },
    sessionsLast24Hours: {
      description: 'Number of unique sessions in the last 24 hours',
      type: 'number',
    },
    isOwner: {
      description: 'Whether the current user has admin/owner role',
      type: 'boolean',
    },
  }}
/>

#### Subscription Object

<TypeTable
  type={{
    monthlyEventCount: {
      description: 'Total events used this month',
      type: 'number',
    },
    eventLimit: {
      description: 'Monthly event limit for the subscription',

      type: 'boolean',

      description: 'Subscription plan name',
      type: 'string',

      description: 'Subscription status',
      type: 'string',
    },

      description: 'Whether this is a pro plan',
      type: 'boolean',
    },
  }}
/>

## Add Organization Member

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
POST /api/organizations/:organizationId/members
```

Adds a user to an organization with a specified role. The user must already exist in the system. Requires admin or owner role in the organization.

### Path Parameters

<TypeTable
  type={{
    organizationId: {
      description: 'Organization ID',
      type: 'string',
      required: true,
    },
  }}
/>

### Request Body

<TypeTable
  type={{
    email: {
      description: 'Email address of the user to add',
      type: 'string',
      typeDescription: 'Must match an existing user',
      required: true,
    },
    role: {
      description: 'Role to assign to the user',
      type: 'string',
      typeDescription: '"admin" | "member" | "owner"',
      required: true,
    },
  }}
/>

### Response

Returns a success message when the user is added successfully.

<TypeTable
  type={{
    message: {
      description: 'Success message',
      type: 'string',
    },
  }}
/>

### Error Responses

<TypeTable
  type={{
    '400': {
      description: 'Invalid input (missing fields, invalid role, or user already a member)',
      type: 'object',
    },
    '401': {
      description: 'Unauthorized (not an admin/owner of the organization)',
      type: 'object',
    },
    '404': {
      description: 'User not found',
      type: 'object',
    },
    '500': {
      description: 'Internal server error',
